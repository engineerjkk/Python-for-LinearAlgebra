* 쿼터니온 이해했다.
1. 여기서 말하는 세개의 i는 허수가 맞다. 그 허수의 의미는 vector이다.
2. 그리고 실수는 scalar로 크기이다. 
3. 여기서 최댓값은 1이며 최솟값은 0인데 1은 원점을 의미하며 0은 원의 표면을 말한다.
4. 그러다가 원 표면을 뚫고나가면 실수축은 음수가 된다.
